## Spring JDBC

This project covers essential Spring JDBC concepts and practical database operations. You will learn:

- Basics of Spring JDBC and database setup  
- Inserting data into the database using Spring JDBC  
- Updating data in the database  
- Deleting data from the database  
- Using RowMapper to map database rows to Java objects  
- Selecting single objects from the database  
- Selecting multiple objects from the database  
- Spring JDBC configuration without XML (Java-based configuration)  
- Autowiring and dependency injection with Spring JDBC  
- Integrating Spring JDBC with real database operations
